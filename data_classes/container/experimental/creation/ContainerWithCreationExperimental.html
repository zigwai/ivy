  • alpha (Optional[float]) ‚Äì The coefficient alpha in the hamming window equation (default: 0.54)

  • beta (Optional[float]) ‚Äì The coefficient beta in the hamming window equation (default: 0.46)

  • dtype (Optional[Union[Array, NativeArray]]) ‚Äì data type of the returned arrays. (default: None)

  • out (Optional[Container]) ‚Äì optional output container, for writing the result to. (default: None)

Return type

Container

Returns

ret ‚Äì The container that includes the Hamming windows.

Examples

>>> x.hamming_window(periodic=True, alpha=0.2, beta=2)

    b: ivy.array([-1.8000, -0.4180,  1.8180,  1.8180, -0.4180])
}
