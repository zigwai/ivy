Parameters

  • self (Array) ‚Äì First input array.

  • x2 (Array) ‚Äì Second input array.

  • rtol (Optional[float]) ‚Äì The relative tolerance parameter. (default: 1e-05)

  • atol (Optional[float]) ‚Äì The absolute tolerance parameter. (default: 1e-08)

  • equal_nan (Optional[bool]) ‚Äì Whether to compare NaN‚Äôs as equal. If True, NaN‚Äôs in a will be (default: False) considered equal to NaN‚Äôs in b in the output array.

  • out (Optional[Container]) ‚Äì Alternate output array in which to place the result. (default: None) The default is None.

Return type

bool

Returns

ret ‚Äì Returns True if the two arrays are equal within the given tolerance; False otherwise.

Examples

>>> x1 = ivy.array([1e10, 1e-7])
>>> x2 = ivy.array([1.00001e10, 1e-8])
>>> y = x1.allclose(x2)
>>> print(y)
False

>>> x1 = ivy.array([1.0, ivy.nan])
>>> x2 = ivy.array([1.0, ivy.nan])
>>> y = x1.allclose(x2, equal_nan=True)
>>> print(y)
True

>>> x1 = ivy.array([1e-10, 1e-10])
>>> x2 = ivy.array([1.00001e-10, 1e-10])
>>> y = x1.allclose(x2, rtol=0.005, atol=0.0)
>>> print(y)
